# sorting_Records_VBA

## Overview

This project demonstrates the use of **VBA (Visual Basic for Applications)** to sort records in Microsoft Excel while providing an interactive user experience through various VBA features.

The application allows users to sort worksheet data efficiently and includes user interaction mechanisms such as message alerts, input prompts, procedure calls, and formatted messages.

---

## Features

### Record Sorting
- Sort records automatically using VBA procedures.
- Supports sorting data based on selected columns.
- Improves data organization and readability.

### Message Box Alerts (`MsgBox`)
- Displays informative messages to users.
- Provides confirmation before performing actions.
- Shows success and error notifications.

### User Input (`InputBox`)
- Prompts users to enter sorting preferences.
- Allows dynamic selection of columns or criteria.

### Procedure Calling
- Demonstrates calling one VBA procedure from another.
- Encourages modular and reusable code structure.

### Formatted Multi-Line Messages (`vbCrLf`)
- Uses `vbCrLf` to display messages across multiple lines.
- Improves readability of user notifications.

## Technologies Used

- Microsoft Excel
- VBA (Visual Basic for Applications)
